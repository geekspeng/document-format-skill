# document-skill

Claude Code Skill 集，专注于文件处理相关的自动化任务。

## 安装

首先添加插件市场：

```bash
/plugin marketplace add geekspeng/document-format-skill
```

然后安装需要的 skill：

```bash
/plugin install document-format@document-skill
```

## 可用 Skills

| Skill | 描述 |
|-------|------|
| [document-format](./skills/document-format/) | 中文文档格式化工具，基于《中文技术文档写作规范》 |
